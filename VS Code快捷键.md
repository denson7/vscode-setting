## VS Code 快捷键

```
// 通用快捷键
Ctrl + C 复制
Ctrl + V 粘贴
Ctrl + X 剪切
Ctrl + F 查询
Ctrl + H 替换
Ctrl + Z 撤销
Ctrl + / 行注释
Ctrl + Enter 在当前行下插入新的一行
Ctrl + 鼠标左键 代码跳转
Alt + Left/Right 向后,向前
Ctrl + G 跳转xx行 # 高频

// 常用
Ctrl + D 匹配当前选中的内容,即选中下一个相同的内容  # 高频
Ctrl + Shift + L 同时选中所有匹配项 # 高频
Ctrl + Shift + P, F1 展示全局命令面板
Shift + Alt + F 代码格式化  # 高频
Shift + Alt +A  注释代码区块
Alt + Up / Down 快速移动行上下  # 高频
Shift + Alt + Up 或 Shift + Alt + Down 快速复制当前行(类似JetBrains的 Ctrl + D)  # 高频
Ctrl + P 跳转文件 # 高频

Ctrl + N  在当前编辑器新建一个文件
Ctrl + Shift + N 打开一个新的编辑器窗口
Ctrl + Shift + W 关闭编辑器
Ctrl +  W 关闭当前窗口
Ctrl + Shift + [ 折叠区域代码

Ctrl + Shift + ] 展开区域代码

Ctrl + [ 代码向左缩进
Ctrl + ] 代码向右缩进
Ctrl + Tab  窗口标签页之间切换

Ctrl + | 拆分窗口 (Ctrl + 1/2/3 切换焦点在不同的切割窗口)
Alt + Shift + 鼠标左键，Ctrl + Alt + Down/Up 多行编辑
Ctrl + K 再按 Ctrl + 2 折叠所有代码(折叠所有方法，2为折叠级别0,1,2,3,4,5,6,7)  # 高频
Ctrl + K 再按 Ctrl + J 展开所有代码

// 界面显示快捷键
Ctrl + B 侧边栏显示隐藏
Ctrl + =|- 放大, 缩小
Ctrl + Shift + F 打开全局搜索
Ctrl + Shift + G 打开Git可视管理
Ctrl + Shift + D 打开DeBug面板

Ctrl + Shift + X 打开插件市场面板

Ctrl + Shift + H 在当前文件替换查询替换

Ctrl + F4, Ctrl + W 关闭编辑器的当前Tab项

```
